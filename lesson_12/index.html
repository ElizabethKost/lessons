Позиціонування

Різне позиціонування CSS дуже часто використовується у верстці сучасних макетів і відкриває для нас нові цікаві можливості для розташування елементів у потоці. Але вам потрібно бути обережним і вміти грамотно впоратися з цими можливостями. Розглянемо такі властивості позиціонування CSS та їх значення як position, top, left, right, bottom та z-index.
CSS властивість position + (left, right, top, bottom і z-index) - встановлює спосіб розміщення елемента відносно вікна браузера або інших об'єктів на веб -сторінці. Допоміжні властивості left, right, top і bottom керують положенням елемента, а z-index керує перекриттям один одного вздовж осі Z.
