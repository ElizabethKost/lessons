CSS transform. 2D и 3D трансформації

2D трансформації

Використовуючи перетворення 2D та 3D, ми можемо змістити, повертати, масштабувати елементи та робити набагато більше! Перетворення з властивістю transform та встановлено такі значення:

Translate (x, y)

Переміщує елемент на нове місце, рухаючись відносно початкового положення об'єкта праворуч і вниз, використовуючи координати X і Y. Далі записується: translate(x, y) або translateX(x, y).
:translate (20px, 20px)
:translate (50%, -20%)

Scale (x, y)

Масштабує елементи, роблячи їх більшими або меншими. Далі записується: scale(x, y) або scale(n).
:scale (0.8, 0.5)
:scale (1.2)
mirror
:scale (-1, 1)

Rotate (n + deg)

Повертає елементи. Далі записується: rotate(n + deg) або rotateX(n + deg).
:rotate (30deg)
:rotate(-45deg)
:rotateX (-70deg)

Skew (x + deg, y + deg)

Деформує (розтягує) елементи. Далі записується: skew(x + deg, y + deg).
:skew(15deg, 15deg)
:skew(0deg, 15deg)
:skew(20deg)

Matrix (a, c, b, d, x, y)

Метод matrix() приймає шість параметрів, що містять математичні функції, які дозволяють обертати, масштабувати, переміщувати (переводити) та нахиляти елементи.
Параметри по порядку слідування: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()). Далі записується: matrix (a, c, b, d, x, y).
:matrix (0.8, -0.5, 0.3, 0.7, 15, 10)
transform: scale(0.8, 0.7) skew(20.56deg, -35.54deg) translate(10.6px, 22px)

Transform-origin

Дозволяє змінити центр трансформації, відносно якого відбувається зміна положення/розміру/форми елемента. Далі записується: transform-origin:top (%)left (%).
transform-origin: 50% (за замовч.)
transform-origin: 0 50%
transform-origin: top left

3D трансформації

Perspective, perspective-origin

Встановлення глибини та зміни початку координати. Не може бути від'ємним значенням.
perspective: 0; transform: rotateX(45deg)
perspective: 500px; transform: rotateX(45deg)
perspective: 400px; perspective-origin: 0 0; transform: rotateX(45deg)

Translate3d (x, y, z)

За аналогією зі своїм 2D братом, це значення встановлює рух елемента, але в 3D -просторі. Окрім осі X і Y, додається вісь Z.
perspective: 500px;
:translate3d (0px, 0px, 0px)
perspective: 500px;
:translate3d (20px, 20px, 50px)

Scale3d (x, y, z)

Встановлює тривимірне масштабування вздовж однієї з осей x y z. Насправді він повторює функціональність свого 2D аналога.
perspective: 500px;
:scale3d (1, 1, 1)
perspective: 500px;
:scale3d (1.2, 1.6, 5)

Rotate3d (x, y, z, deg)

Обертає елемент відносно трьох осей. Елемент обертається під кутом, встановленим останнім параметром відносно напрямку напрямку X, Y, Z.
perspective: 500px;
:rotate3d (1, 1, 1, 0deg)
perspective: 500px;
:rotate3d (1, 1, 1, 60deg)

Matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

Встановлює тривимірну трансформацію як однорідну матрицю з розміром 4 на 4 з шістнадцятьма значеннями у стовпцях.

Множині 3D трансформації

Дозволяє поєднувати кілька 3D -перетворень в одному.
:translate3d (20px, 30px, 10px) rotate3d (1,1,1,30deg)

Transform-style

Визначає, як дочірні елементи повинні відображатися в просторі 3D.
perspective: 500px;
transform-style: preserve-3d

Backface-visibility

Відображення задньої частини об'єкта.
backface-visibility: hidden
