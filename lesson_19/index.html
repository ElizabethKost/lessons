fr - специфічна одиниця довжини, одиниця гнучкості. Відмінність і плюси fr перед %, приклад:
У нас є контейнер, в якому 5 div. Щоб рівномірно розділити ширину 100% між кожним div, нам потрібно вказати індивідуальну ширину для кожного в 20% (100% / 5 div = 20%).
Але, якщо ще додається один div, то нам доведеться все перерахувати (100% / 6 div = 16,66%)!
Саме цю проблему вирішує fr — одиниця гнучкості. Якщо вказати ширину в 1fr, то ширина кожного елемента буде рівномірно розділена між дочірніми. Приклад:
Загальна частка = 1.5fr + 3fr + 4.5fr = 9fr
Якщо батьківський контейнер має ширину 900px
I значення = 1.5fr / 9fr * 900px = 150px.
II значення = 3fr / 9fr * 900px = 300px.
III значення = 4.5fr / 9fr * 900px = 450px.

Властивість .grid-контейнера
{display: grid;
grid-template-columns: 1.5fr 3fr 4.5fr;}

Побудова неявної сітки

Якщо елемент сітки розташований у лінії або стовпці, розмір яких не визначається явною сіткою, створюються неявні доріжки сітки для їх відображення. Це може статися, якщо рядок або стовпець знаходяться поза встановленим розміром сітки.

Елементи сітки, які не розміщуються в явній частині, автоматично розміщуються в незайнятому просторі в контейнері-сітці, використовуючи алгоритм автоматичного розміщення. Властивість сітки grid-auto-flow контролює автоматичне розміщення елементів сітки без явного положення. Після заповнення явної сітки (або якщо немає явної сітки), автоматичне розміщення також призведе до генерації неявної частини сітки. Властивість не успадковується.

Властивість grid-auto-columns та grid-auto-flow: column

Grid-auto-columns - дозволяє керувати розміром стовпців неявної частини сітки. Якщо наведено кілька розмірів частини, шаблон повторюється за необхідності, щоб знайти розмір неявних частин. Перша неявна частина сітки після явної сітки отримує перший заданий розмір і т.д. Властивості не успадковуються.

Grid-auto-flow: column - автоматично розміщує елементи, заповнюючи кожен стовпець по черзі зліва на право, додаючи нові стовпці за необхідності.

Властивість .grid-контейнера
{display: grid;
grid-template: 1fr / repeat(3,200px);
grid-auto-flow: column
grid-auto-columns: 50px;}

Grid-area та grid-template-areas

Grid-area - це універсальна властивість, для скороченого запису:
grid-row-start (задає першу горизонтальну лінію для елемента);
grid-column-start (задає першу вертикальну лінію для елемента);
grid-row-end (задає останню горизонтальну лінію для елемента);
grid-column-end (задає останню вертикальну лінію для елемента)

Властивість .grid-контейнера
{display: grid;
grid-template: repeat(3,100px) / repeat(3, 1fr);}

Властивість II .grid-елемента
{grid-area: 2 / 2 / 4 / 4;}
